---
title: 'Instalação'
description: 'Guia passo a passo para configurar e executar a API Hunno CRM em um ambiente de desenvolvimento local.'
---

<Info>
  **Pré-requisitos**:
  - Node.js
  - Git instalado na sua máquina
  - Um banco de dados PostgreSQL em execução
</Info>

Siga os passos abaixo para clonar, configurar e executar o projeto localmente.

<Steps>
  <Step title="1. Clone o Repositório">
    Use o Git para baixar o código-fonte do projeto.
    ```bash
    git clone https://github.com/marciopollo/hunnocrm-api-node
    ```
  </Step>

  <Step title="2. Acesse o Diretório">
    Navegue até a pasta do projeto que foi criada.
    ```bash
    cd hunnocrm-api-node
    ```
  </Step>

  <Step title="3. Instale as Dependências">
    Execute o comando para instalar todas as bibliotecas e pacotes que a API necessita.
    ```bash
    npm install
    ```
  </Step>

  <Step title="4. Configure as Variáveis de Ambiente">
    O projeto precisa de um arquivo `.env` para informações sensíveis. Copie o arquivo de exemplo para criar o seu.
    ```bash
    cp .env.example .env
    ```
    Após copiar, abra o arquivo `.env` e preencha a variável `DATABASE_URL` com a string de conexão do seu banco de dados PostgreSQL.
  </Step>

  <Step title="5. Execute as Migrações do Banco de Dados">
    Este comando irá criar todas as tabelas necessárias no seu banco de dados.
    ```bash
    npm run migrate:dev
    ```
  </Step>

  <Step title="6. Inicie o Servidor de Desenvolvimento">
    Com tudo configurado, execute o comando para iniciar a API.
    ```bash
    npm run dev
    ```
    O servidor será iniciado, geralmente na porta `3333`.
  </Step>
</Steps>
