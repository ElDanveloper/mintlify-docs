---
title: 'Instalação'
description: 'Guia passo a passo para configurar e executar a API Hunno CRM em um ambiente de desenvolvimento local.'
---

<Info>
  **Pré-requisitos**:
  - Node.js
  - Git instalado na sua máquina
  - Um banco de dados PostgreSQL em execução
</Info>

Siga os passos abaixo para clonar, configurar e executar o projeto localmente.

<Steps>
<Step title="Clone o Repositório">

```bash
git clone https://github.com/marciopollo/hunnocrm-api-node
```
</Step>

<Step title="Acesse o Diretório">

Navegue até a pasta do projeto que foi criada.

```bash
cd hunnocrm-api-node
```
</Step>
</Steps>

<Step title="Instale as Dependências"> Execute o comando abaixo para instalar todas as bibliotecas e pacotes que a API necessita para funcionar.

Bash

npm install
</Step>

<Step title="Configure as Variáveis de Ambiente"> O projeto precisa de um arquivo .env para armazenar informações sensíveis, como a conexão com o banco de dados. Copie o arquivo de exemplo para criar o seu.

Bash

cp .env.example .env
Após copiar, abra o arquivo .env com um editor de texto e preencha a variável DATABASE_URL com a string de conexão do seu banco de dados PostgreSQL. </Step>

<Step title="Execute as Migrações do Banco de Dados"> Este comando irá criar todas as tabelas e estruturas necessárias no seu banco de dados, com base no esquema definido no Prisma.

Bash

npm run migrate:dev
</Step>

<Step title="6. Inicie o Servidor de Desenvolvimento"> Com tudo configurado, execute o comando abaixo para iniciar a API.

Bash

npm run dev
O servidor será iniciado e a API estará pronta para receber requisições, geralmente na porta 3333. Você verá uma mensagem de confirmação no terminal. </Step> </Steps>
